## About This Project

The Bracero Program is a historical moment in American-Mexican relations as agricultural lobbies pushed legislators to create a program that would enable Mexican migrants fill in a supposed shoratge in labor due to the war effort. To learn more about the Bracero Program, and the historical context surrounding it, be sure to check out my website.

This repository holds the code contents for the final creative project assignment for the Dickinson College course LALC298. If you choose to fork this repo, please provide proper acknowledgements.
